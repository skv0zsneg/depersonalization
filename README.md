# depersonalization

Выпускная квалификационная работа на тему "Исследования обезличивания данных в информационной системе университета"

___
### Запуск
```bash
sudo docker-compose -f docker-compose.yaml up --remove-orphans
python3 run.py
```
___
### Использованные технологии:
- Python 3
- Flask
